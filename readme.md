# 全國超級摸摸貓貓競賽

原：[高科盃全國商業智慧競賽](https://cpt.ic.nkust.edu.tw)

現：https://projects.rainmeocat.com/cpt-pet-cats

為避免誤導，已把內容全部撤換為貓咪。

使用雙版面的設計方法`pc.vue`和`mobile.vue`

以navigator.userAgent在router切換電腦版和手機版。

![](https://i.imgur.com/uRhTr5w.jpg)

## 專案依賴安裝
```
npm install
```

### 開發
```
npm run serve
```

### 打包
```
npm run build
```

### Lint和自動修正
```
npm run lint
```
